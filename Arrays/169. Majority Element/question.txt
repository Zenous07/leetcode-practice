169. Majority Element

Difficulty: Easy

Problem Statement:
Given an array nums of size n, return the majority element.

The majority element is the element that appears more than ⌊n / 2⌋ times. You may assume that the majority element always exists in the array.

---

Example 1:
Input: nums = [3, 2, 3]
Output: 3

Explanation:
The number 3 appears 2 times, which is more than ⌊3 / 2⌋ = 1.
So, the majority element is 3.

---

Example 2:
Input: nums = [2, 2, 1, 1, 1, 2, 2]
Output: 2

Explanation:
The number 2 appears 4 times, which is more than ⌊7 / 2⌋ = 3.
So, the majority element is 2.

---

Constraints:
1 <= nums.length <= 5 * 10^4
-10^9 <= nums[i] <= 10^9
The input is generated such that a majority element always exists in the array.

---

Follow-up:
Could you solve the problem in linear time and in O(1) extra space?
