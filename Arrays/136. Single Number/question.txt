136. Single Number

Difficulty: Easy

Problem Statement:
Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.

You must implement a solution with linear runtime complexity and use only constant extra space.

---

Example 1:
Input: nums = [2, 2, 1]
Output: 1

Explanation:
The number 2 appears twice, and the number 1 appears only once.
So, the single number is 1.

---

Example 2:
Input: nums = [4, 1, 2, 1, 2]
Output: 4

Explanation:
The numbers 1 and 2 appear twice, and the number 4 appears only once.
So, the single number is 4.

---

Example 3:
Input: nums = [1]
Output: 1

Explanation:
There is only one element in the array, so it is the single number.

---

Constraints:
1 <= nums.length <= 3 * 10^4
-3 * 10^4 <= nums[i] <= 3 * 10^4
Each element in the array appears exactly twice except for one element which appears only once.
