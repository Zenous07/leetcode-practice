1189. Maximum Number of Balloons  
Difficulty: Easy  

Problem Statement: Given a string text, you want to use the characters of text to form as many instances of the word "balloon" as possible.  

You can use each character in text at most once. Return the maximum number of instances that can be formed.  

---  

Example 1:  
Input: text = "nlaebolko"  
Output: 1  

Explanation: The string can form one instance of "balloon".  

---  

Example 2:  
Input: text = "loonbalxballpoon"  
Output: 2  

Explanation: The string can form two instances of "balloon".  

---  

Example 3:  
Input: text = "leetcode"  
Output: 0  

Explanation: It is not possible to form the word "balloon".  

---  

Constraints:  
1 <= text.length <= 10^4  
text consists of lower case English letters only.
