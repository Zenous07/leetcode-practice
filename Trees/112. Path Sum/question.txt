112. Path Sum

Difficulty: Easy

Problem Statement:
Given the root of a binary tree and an integer targetSum, return true if the tree has a root-to-leaf path such that adding up all the values along the path equals targetSum.

A leaf is a node with no children.

---

Example 1:
Input: root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22
Output: true

Explanation:
The root-to-leaf path (5 → 4 → 11 → 2) has a sum of 22, which equals the targetSum.

---

Example 2:
Input: root = [1,2,3], targetSum = 5
Output: false

Explanation:
There are two root-to-leaf paths:
(1 → 2): Sum = 3
(1 → 3): Sum = 4

Neither path has a sum equal to 5, so the answer is false.

---

Example 3:
Input: root = [], targetSum = 0
Output: false

Explanation:
The tree is empty, so there are no root-to-leaf paths.

---

Constraints:
0 <= The number of nodes in the tree <= 5000
-1000 <= Node.val <= 1000
-1000 <= targetSum <= 1000
