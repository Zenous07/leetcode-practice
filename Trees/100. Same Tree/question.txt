100. Same Tree

Difficulty: Easy

Problem Statement:
Given the roots of two binary trees p and q, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.

---

Example 1:
Input: p = [1,2,3], q = [1,2,3]
Output: true

Explanation:
Both binary trees have the same structure and the same node values.
So, the trees are identical.

---

Example 2:
Input: p = [1,2], q = [1,null,2]
Output: false

Explanation:
The structures of the two trees are different.
Therefore, they are not the same tree.

---

Example 3:
Input: p = [1,2,1], q = [1,1,2]
Output: false

Explanation:
Although the structures are the same, the node values differ.
Therefore, the trees are not identical.

---

Constraints:
The number of nodes in both trees is in the range [0, 100].
-10^4 <= Node.val <= 10^4
